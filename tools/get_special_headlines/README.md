# get_special_headlines

## Description
Access curated news themes such as 'Economy', 'Stocks', 'Sports', or 'Toys'. 

## Interface
- **Input**: `category` (string), `limit` (default: 5)
- **Output**: Array of themed news objects.

## Showcase Note
This tool leverages specialized thematic scrapers and domain-specific normalization rules in the private engine.
