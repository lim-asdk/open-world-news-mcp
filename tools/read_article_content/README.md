# read_article_content

## Description
Deep-reads the main text of a news article. This tool is a core part of the "Active AI Agent" strategy, allowing models to verify facts and summarize content directly from the source.

## Interface
- **Input**: `url` (valid news URL)
- **Output**: Pure article text content.

## Showcase Note
The production engine uses advanced heuristics to bypass clutter, ads, and common paywall elements.
